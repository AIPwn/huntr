## [0.1.0] - 2024-04-02

### 新增
- 初始版本发布
- 支持 7 种漏洞类型检测
- 集成 LLM 分析能力
- 提供命令行界面
- Docker 支持

### 漏洞类型支持
- 本地文件包含 (LFI)
- 任意文件覆盖 (AFO)
- 远程代码执行 (RCE)
- 跨站脚本 (XSS)
- SQL 注入 (SQLI)
- 服务器端请求伪造 (SSRF)
- 不安全的直接对象引用 (IDOR)

### 功能特性
- 自动代码分析
- 漏洞检测
- 置信度评分
- PoC 生成
- 详细报告输出

### 依赖
- Python 3.10
- Claude/GPT API
- Jedi 解析器
